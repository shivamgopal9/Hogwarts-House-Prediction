# Hogwarts House Prediction

## Project Overview
This project predicts a student's Hogwarts House using a Deep Neural Network built with TensorFlow and Keras.

## Dataset
- 1000 samples
- 10 features
- 4 target classes:
  - Gryffindor
  - Hufflepuff
  - Ravenclaw
  - Slytherin

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow/Keras
- Matplotlib
- Seaborn

## Results
- Accuracy: 99.50%
- Precision: 99.51%
- Recall: 99.50%
- F1-Score: 99.50%

## Model Architecture
- Dense(128) + ReLU
- BatchNormalization
- Dense(64) + ReLU
- Dense(16) + ReLU
- Dense(4) + Softmax
